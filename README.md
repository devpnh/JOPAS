# JOPAS s.r.o. — web

Jednostránkový web pre **JOPAS s.r.o.** — vodár a kúrenár, Michalovce a okolie.
Voda, kanalizácia, kúrenie, kotly a sanita.

## Stack

- React 18 + Vite 5
- Tailwind CSS 3
- Framer Motion (decentné reveal animácie)
- lucide-react (ikony)

Vizuálny smer: jemne industriálny — betón/grafit + meď, characterful grotesk
(Bricolage Grotesque) + Inter + JetBrains Mono, „spec-sheet" hlavičky sekcií,
telefón ako hlavné CTA.

## Vývoj

```bash
npm install
npm run dev        # http://localhost:5180
npm run build      # produkčný build do dist/
npm run preview    # náhľad buildu
```
